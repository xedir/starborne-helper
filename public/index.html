Hex properties:

Use resource buttons to add resources to hexes via click.

Resources are displayed on hexes as [metal, gas, crystal, labor], a hex with [2,2,0,3] has 2 metal, 2 gas, 0 crystal and 3 labor.

Build a station (claims 4 range), mining facility (MF) or hab dome (HD) with the according buttons, can only be built on claimed hexes.

Export generates export text at the bottom of the page, load takes export text as input and generates the map.
